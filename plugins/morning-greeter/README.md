# Morning Greeter

Sends a motivational morning greeting via Slack DM to Will. The message is generated fresh each time — warm, direct, and in Spanish — drawing from themes like growth, discipline, and building something meaningful. Can be triggered on a schedule or on demand.

## Install

```
/plugin install morning-greeter@singular-agency-marketplace
```

## Skills

| Skill | Trigger phrases | What it does |
|-------|----------------|--------------|
| `/morning-greeter` | "send morning greeting", "saludo de la mañana", "buenos días slack", "morning greeting", "send my daily motivation", "manda el saludo" | Generates a unique motivational message in Spanish and sends it as a Slack DM to Will |

## Requirements

- Slack MCP connected with DM permissions
