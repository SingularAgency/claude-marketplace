# Welcome Email Drafter

Redacta emails de bienvenida personalizados para nuevos clientes de Singular Agency. Busca datos del cliente en Airtable, genera el email en español o inglés con tono profesional cálido, y crea un borrador listo en Gmail. Incluye presentación de Miguel, el equipo, servicios contratados, próximos pasos y canales de comunicación.

## Install

/plugin install welcome-email-drafter@singular-agency-marketplace

## Skills

| Skill | Trigger phrases | What it does |
|-------|----------------|--------------|
| /welcome-email-drafter | "Manda un email de bienvenida a [cliente]", "Draft a welcome email for [client]", "Preséntame con el cliente nuevo" | Drafts a personalized onboarding welcome email, looks up client info in Airtable, and creates a Gmail draft ready to send |
