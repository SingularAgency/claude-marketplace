# Meeting Message Drafter

Fetches meeting transcripts from Airtable and drafts any message, follow-up, or summary — for clients or the internal team — based on what was actually discussed.

## Install

```
/plugin install meeting-message-drafter@singular-agency-marketplace
```

## Requirements

- Airtable MCP connector must be enabled in your Cowork session.
- Gmail or Slack MCP connector (optional, for sending drafted messages directly).

## Skills

| Skill | What it does |
|---|---|
| `meeting-message-drafter` | Reads a meeting transcript from Airtable and drafts a message for any recipient based on the user's intent |

## Example triggers

- "En la reunión de Velvet de hoy quedé de mandarle un mensaje a Maxi..."
- "Redacta un follow-up para el cliente de la reunión de Bloomer de ayer"
- "Qué acciones quedaron pendientes de la reunión de Ediphi?"
- "A partir de la reunión de hoy, escribe un resumen para el equipo"
